Ferramentas para participar do Remote Dojo

Para participar remotamente será preciso ter as seguintes ferramentas instaladas:

Visual Studio Code

O Visual Studio Code é o editor de código free, open source e multiplataforma da Microsoft, se ainda não conhece acesse: https://code.visualstudio.com/.

Plug-in VS Live Share

O VS Live Share é uma extensão para o VS Code, que permite mais de uma pessoa acessar remotamente o seu código de maneira interativa no editor! Para instalar acesse o Marketplace. Veja esse vídeo introdutório para aprender mais.

Hangout

Se você não estiver como piloto ou co-piloto, acompanhe através do Hangout até chegar a sua vez de participar.
